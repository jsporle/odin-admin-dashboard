# odin-admin-dashboard
Full brief available at https://www.theodinproject.com/lessons/node-path-intermediate-html-and-css-admin-dashboard


2026.04.14

Things I've tried out during this project:

- Combining grid and flex boxes.
- Mapping grid areas (row-start / col-start / row-end / col-end)
- Using clamp to scale text based on viewport width
- Using Google Fonts Material Symbols for icons
- Styling cursor changes and animations for hover and active links and buttons etc.
- Using @keyframes to customise animation loops
- Styling with :not to exclude elements
- Commenting out coding (Ctrl + /) to debug overflow issues
- Multiline editing (alt + left click, Ctrl + shift + A to enter Emmet abbreviations)
- Nesting adjustment (select block and then Tab to indent or shift+Tab to outdent)
- Ctrl + shift + A and "lorem" to generate dummy text
